## Preparation

Before running the project add your rabbitMQ server address in the 'src\rabbitMQ\rabbitMQ.service.ts' file and the MongoDB address in the 'src\app.module.ts' file 

```bash
$ npm i
```

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm start
```

## Test

```bash
# unit tests
$ npm test
```
